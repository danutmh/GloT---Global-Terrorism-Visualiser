Getting Started

Welcome to the GloT Help page. Here you will find guidance on how to navigate and use the various features of the Global Terrorism Visualiser effectively.

Navigating the Interface

The main interface consists of a map visualization, search options, and various data representation tools. Use the navigation bar at the top to go to various places.

Using the Map

To interact with the map, you can zoom in/out, pan across different regions, and click on specific markers for detailed information about terrorist incidents.

Understanding Visualizations

The visualizations include heatmaps, charts, and graphs.

Exporting Data

Data can be exported in CSV, PNG, or SVG formats.

Contact

If you encounter any issues while using GloT, please contact us at website@stopterrorism.com
